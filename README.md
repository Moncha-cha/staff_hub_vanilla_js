Asset Management System – Vanilla JavaScript Edition

Tento projekt představuje původní verzi systému pro správu majetku, postavenou na čistém JavaScriptu. Slouží jako ukázka schopnosti pracovat s DOM API, imperativním programováním a manuální správou stavu aplikace.

🚀 Funkcionalita
Imperativní Rendering: Dynamické generování HTML karet pomocí forEach a innerHTML.

Manuální Filtrování: Logika vyhledávání a filtrování navázaná na input a click eventy.

DOM Manipulace: Přímý přístup k elementům pomocí querySelector pro aktualizaci počítadel a stavů.

CSS proměnné: Centrální správa barev a stylů pro snadnou údržbu UI.

🛠 Technický stack
HTML5 – Sémantická struktura dokumentu.

CSS3 – Pokročilé pozicování a responzivita.

Vanilla JavaScript (ES6+) – Práce s poli, řetězcové šablony (template literals), arrow functions.

🧠 Co tento projekt demonstruje
Znalost základů: Schopnost postavit funkční aplikaci bez frameworků.

Event Handling: Efektivní používání addEventListener pro interaktivitu.

Data-to-UI Sync: Pochopení procesu, jak ručně synchronizovat data v paměti s tím, co uživatel vidí na obrazovce.

🇬🇧 Vanilla JS Version (English)
Asset Management System – Vanilla JavaScript Edition
This is the foundational version of the asset management tool, built entirely with Vanilla JavaScript. It demonstrates core proficiency in DOM manipulation, imperative programming patterns, and manual state synchronization.

🚀 Features
Imperative Rendering: Dynamically generating UI components using forEach loops and innerHTML injection.

Manual Filtering & Search: Custom logic for real-time filtering tied to native JS event listeners.

Direct DOM Access: Using querySelector and classList to update counters and UI states manually.

CSS Variables: Consistent styling and easy theming through native CSS custom properties.

🛠 Tech Stack
HTML5 – Semantic markup and structure.

CSS3 – Layout design using Flexbox and custom styling.

Vanilla JavaScript (ES6+) – Array methods, Template Literals, and modern JS syntax.

🧠 Core Competencies Demonstrated
Fundamental Mastery: Building complex interactive features without external libraries or frameworks.

Event Management: Handling user interactions via the native Event Loop and listeners.

Manual Reconciliation: Understanding how to manually bridge the gap between the data model and the View layer.
